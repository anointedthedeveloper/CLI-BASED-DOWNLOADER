AnimePahe Downloader — FlareSolverr Folder
==========================================

Place the FlareSolverr executable and its 'internal' folder here:

    fs/
        flaresolverr.exe    ← the main executable
        internal/           ← required companion folder (comes with the release)
            ...

Download the latest FlareSolverr release (windows-x64.zip) from:
    https://github.com/FlareSolverr/FlareSolverr/releases

Extract the zip and copy flaresolverr.exe + internal/ into this fs/ folder.
The app will auto-launch FlareSolverr on startup once the exe is present here.

You can also test FlareSolverr standalone by running:
    python fs_launcher.py
